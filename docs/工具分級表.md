# ERP 專題 工具分級表

> Day 1 上午交付文件  
> 所有 AI Agent 呼叫工具都必須通過 Tool Gateway，Gateway 依此表判斷要直接執行還是攔截送審。

---

## 風險等級說明

| 等級 | 說明 | Gateway 行為 |
|------|------|-------------|
| 🟢 `read_only` | 只查詢，不改資料庫 | 直接執行 |
| 🟡 `suggestion` | 產生建議，不寫入資料庫 | 直接執行 |
| 🔴 `write` | 會改資料庫 | 攔截，送審批後才執行 |
| ⛔ `dangerous` | 刪除、大量修改等高風險 | 攔截，需審批＋二次確認 |

---
